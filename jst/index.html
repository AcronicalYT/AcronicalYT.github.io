JST Discord Bot

Why not check out my new Discord bot? JST is a new moderation/utility discord bot made by me that can help with setting up a server, moderating, and more. It is currently in development and I am working on making it more user-friendly and more useful. I hope you enjoy using the bot and I would love to hear feedback about it!

You can invite the bot from the support server here!

Below you can find a list of commands avaliable with the bot, these range from moderation to utility commands. If you have any questions or concerns about the bot, feel free to join the support server, and I will be happy to help you!

Commands

General

help - Displays all commands
ping - Displays the bot's ping
info - Displays information about the bot
invite - Invite the bot to your server
support - Get a link to the support server
report-bug - Report a bug
suggest - Suggest a feature
steal-emoji - Steal an emoji from another server

Moderation

ban - Bans a user
kick - Kicks a user
mute - Mutes a user
temp-mute - Temporarily mutes a user
unmute - Unmutes a user
purge - Purges a certain amount of messages
warn - Warns a user
clear-warns - Clears all warnings for a user
list-warns - Lists all warnings for a user
lock - Lock the server
unlock - Unlock the server
force-nick - Force a user's nickname

Fun

hangman - Play hangman with another user
song-lyrics - Get the lyrics of a song

Utilities

create-channel - Create a channel
delete-channel - Delete a channel
add-role - Add a role to a user
remove-role - Remove a role from a user
nick - Change your nickname
translate - Translate text into another language
ticket - Creates a ticket

Setup

set-prefix - Set the prefix for the bot
set-log-channel - Set the log channel for the bot
set-welcome - Set the welcome channel for the bot
set-admin-role - Set the admin role for the bot
set-mod-role - Set the mod role for the bot
set-mute-role - Set the mute role for the bot
reset-prefix - Reset the prefix for the bot

Advanced Roles

add-autorole - Add a role to the autorole list
remove-autorole - Remove a role from the autorole list
clear-autorole - Clear the autorole list
add-reaction-role - Add a role to the reaction role list
remove-reaction-role - Remove a role from the reaction role list
panel - Sends the reaction role selector panel
